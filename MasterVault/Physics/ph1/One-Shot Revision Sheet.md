## 📋 5-MINUTE FORMULA CRAM SHEET

### Coulomb's Law
```
F = kq₁q₂/r²  [k = 9×10⁹]
Vector: F₂₁ = (kq₁q₂/r²₁₂)r̂₂₁
In medium: F = F_vacuum/K
```

### Electric Field
```
E = F/q = kQ/r²
Force: F = qE
E_parallel_wire = λ/(2πε₀r)
E_infinite_sheet = σ/(2ε₀)
E_shell(outside) = kQ/r²
E_shell(inside) = 0
```

### Dipole
```
p = q×2a (from - to +)
E_axis = 2p/(4πε₀r³)
E_equat = p/(4πε₀r³)
τ = pE sinθ
U = -pE cosθ
```

### Gauss's Law
```
∮E·dS = q/ε₀
Flux = EA (when E ⊥ surface)
```

## 🧠 5-MINUTE CONCEPT CRAM SHEET

### Properties of Charge
- [ ] Quantisation: q = ne (e = 1.6×10⁻¹⁹ C)
- [ ] Conservation: Total charge constant
- [ ] Additivity: Algebraic sum

### Force Rules
- [ ] Like charges: Repel
- [ ] Unlike charges: Attract
- [ ] F ∝ 1/r² (Coulomb)
- [ ] F ∝ 1/r (Wire) - after Gauss's law
- [ ] F ∝ constant (Infinite sheet)

### Field Rules
- [ ] +Q: Field outward
- [ ] -Q: Field inward
- [ ] Dipole: E ∝ 1/r³ (faster decay)
- [ ] E = 0 inside conductor

### Flux Rules
- [ ] + inside: Positive flux
- [ ] - inside: Negative flux
- [ ] 0 inside: Zero flux
- [ ] Shape doesn't matter for net flux

### Dipole Rules
- [ ] Axis field = 2 × Equatorial field
- [ ] Stable: θ = 0° (p parallel to E)
- [ ] Torque max at θ = 90°
- [ ] U minimum when aligned

## 🔑 5-MINUTE MEMORY TRICKS

### "Glass Gives Electrons"
- Glass loses electrons → positive
- Silk gains electrons → negative

### "Field Away, Force Toward"
- Field: Away from positive
- Force on negative: Toward positive (opposite to field)

### "Stable Stands"
- Stable equilibrium → p parallel to E → minimum PE

### "K Reduce Force"
- In medium with dielectric K: F = F₀/K

### "Wire is 1/r"
- Field from wire: E ∝ 1/r (like traffic from road)

### "Shell is Point"
- Outside spherical shell: treat as point charge at center

### "Inside Shell Zero"
- Inside any spherical shell: E = 0 (by symmetry)

### "Flux Don't Care"
- Net flux depends only on enclosed charge
- Shape doesn't matter
