# Definite Integration

Definite integration is central for area, accumulation, and symmetry problems. It links to [[Calculus Core]] and [[Differentiation]].

Key property:

$$\int_a^b f(x)dx = \int_a^b f(a+b-x)dx$$

For even and odd functions:

$$\int_{-a}^{a} f(x)dx = 2\int_0^a f(x)dx$$

when f is even, and zero when f is odd.

- Q:: What is the fastest check before solving a symmetric definite integral? A:: Test whether the function is even, odd, or transformable with f(a+b-x).
- MCQ:: Integral of an odd function from -a to a is: Options:: Zero | Twice integral from 0 to a | Infinite | Always positive Answer:: Zero
